Remarks
The algorithm is: 1. initialize list of contigs to empty list. List of seqs is passed as argument. 2. compute pairwise overlap scores for each pair of input seqs (with reversal and complementation as appropriate). 3. choose best overlap score. the “merge items” (can be seqs or contigs) are the items with that score. If best score is less than threshold, assembly is finished. 4. merge the merge items into a single contig and remove them from their list(s) 5. compute the overlap between new item and all existing items 6. go to step 3
