XsvContigFormatter Class
This class will write a contig as a list of sparse sequences using the XsvSparseFormatter. The first sequence is the consensus, and the rest are the assembled sequences offset from the consensus. E.g. formatting a Contig with 2 assembled sequences, using '#' as sequence prefix and ',' as character separator. # 0,100,Consensus 12,A 29,T 39,G #3,10,Fragment1 9,A #25,20,Fragment2 4,T 14,G

Write(Stream, Contig)
Formats a (sparse) contig to a character-separated value file, writing the consensus first, followed by the sequence separator, and each assembled sequences followed by the sequence separator. The consensus has an offset of 0, while the assembled sequences have the offset as present in AssembledSequence.Position.
