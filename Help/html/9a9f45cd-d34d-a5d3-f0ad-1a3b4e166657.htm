NexusParser Class
A NexusParser reads from a source of text that is formatted according to the NexusParser flat file specification, and converts the data to in-memory ISequenceAlignment objects. For advanced users, the ability to select an encoding for the internal memory representation is provided. There is also a default encoding for each alphabet that may be encountered.
