Properties

IsComplemented
Gets or sets a value indicating whether the sequence was complemented in order to find sufficient overlap.
IsReversed
Gets or sets a value indicating whether the orientation of the sequence was reversed in order to find sufficient overlap.
Remarks
If the assembly algorithm used AssumeStandardOrientation=true, then IsReversed and IsComplemented will both be true (reverse complement) or both be false.
