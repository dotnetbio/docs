GraphScaffoldBuilder
Generates Scaffolds using Graph. Algorithm: Step1: Generate contig overlap graph. Step2: Map Reads to contigs. Step3: Generate Contig Mate Pair Map. Step4: Filter Paired Reads. Step5: Distance Orientation. Step6: Trace Scaffold Paths. Step7: Assemble paths. Step8: Generate sequence of scaffolds.
