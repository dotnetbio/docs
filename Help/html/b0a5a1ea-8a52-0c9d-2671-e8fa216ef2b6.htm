A signed number indicating the relative values of index property of this instance and the index property of specified IndexedItem. Return Value Description: Less than zero index of this instance is less than the index of specified IndexedItem. Zero index of this instance is equal to the index of specified IndexedItem. Greater than zero index of this instance is greater than the index of specified IndexedItem or specified IndexedItem is null.
