SnpParser Class
Implements common methods for parsing SNPs from a SnpReader into ISequences. This reads Snp items from the SnpReader and stores either of the two alleles in a sparse sequence at the same position as the chromosome position. Extending classes have to implement the SnpReader GetSnpReader(TextReader reader) method that returns a SnpReader for the given TextReader.
