ReadContigMap Class
Class stores multiple mapping between reads and a contig. ------------------- Read Sequence ------------------------------ Contig Sequence [Full Overlap] ---------------- Contig Sequence [Partial Overlap] The Class stores Key: Sequence Id of Read Value Key: Sequence of Contig Value: List of position of Overlaps of contig with read.
