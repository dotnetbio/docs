ErodeGraphEnds Method
IGraphEndsEroderErodeGraphEnds Method
Erode ends of graph that have low coverage. For optimization of another step (dangling link purger) in assembly process, this returns a list of integers. In case this optimization is not used, a single element list containing the number of eroded nodes can be returned.
